# 🤖 Mock Interview AI Agent

An AI-powered interview preparation platform that conducts personalized mock interviews, evaluates candidate performance, predicts placement readiness using Machine Learning, and provides actionable feedback to improve interview skills.

---

## 🌐 Live Demo

🔗 **Application:** https://mock-interview-ai-agent-eybx.onrender.com

---

## 🚀 Overview

Mock Interview AI Agent simulates real-world technical and behavioral interviews by analyzing a candidate's resume and generating personalized interview questions. The system dynamically adapts to user responses, evaluates overall performance, predicts placement readiness using Machine Learning, and provides AI-generated feedback to help candidates improve their interview skills.

---

## ✨ Key Features

- 📄 Resume Upload & Intelligent Resume Parsing
- 🤖 AI-Generated Technical and Behavioral Interview Questions
- 💬 Dynamic Follow-up Question Generation
- 🎯 Multiple Interview Modes
  - Friendly Recruiter
  - Busy Senior Engineer
  - Skeptical Interviewer
- 🎤 Voice Input Support
- 📊 AI-Based Interview Performance Evaluation
- 📈 Machine Learning Placement Readiness Prediction
- ✍️ AI-Powered Answer Improvement ("What You Should Have Said")
- 💾 Interview History Management
- ☁️ Fully Deployed Cloud Application
